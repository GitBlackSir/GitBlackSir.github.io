快速幂取模算法

September 20, 2017 . 算法和数据结构

快速幂

  • 快速幂的目的是为了快速求幂
  • 假设我们要求a^b，普通算法是把a连乘b次，时间复杂度为O(b)
  • 经过快速幂优化后时间复杂度为O(logb)

例：a^10
10 = (1010)2 a^10 = a^( 2^1 + 2^3)

&运算用于二进制取位操作，（a&1） 为该二进制数最后一位。 &运算还可以判断奇偶 (a&1) == 0 为偶数， (a&1) == 1 为奇数。 >>运算可以去掉二进制最后一位的。

long long fast(long long a, long long b){
 long long count = 1;//记录
	while(b){
	if (b & 1) //如果二进制为真参与运算
    	count *= a;
	a *= a; //加倍
	b >>= 1; //相当于b/2
	}
return count;
}

快速幂取模

long long fast(long long a, long long b, long long c){
 long long count = 1; //记录
	while(b){
	if (b & 1) //如果二进制为真参与运算
    	count = (count * a) % c;
	a *= a; //加倍
	b >>= 1; //相当于b/2
	}
return count % c;
}
